## 🛰️ O Projekte
Tento projekt demonštruje prepojenie vlastného **Python (Flask)** backendu s dynamickým **JavaScript** frontendom. Pôvodné externé API bolo nahradené vlastným riešením.

### Hlavné funkcie:
* **Custom API:** Dáta sú poskytované z lokálneho servera na porte 5000.
* **Midnight Design:** Moderné tmavé rozhranie šetriace oči.
* **Dynamické Filtrovanie:** Vyhľadávanie študentov v reálnom čase.
